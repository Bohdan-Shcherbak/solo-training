src - папка з ісходними, початковими файлами
gulp - вспоміжні(вспомагательные) файли для роботи з галп
config - файли налаштувань
path - пути различные
plugins - інформація про плагіни, які бдуть використовуватись
ftp - налаштування для вигрузки результату на фтп сервер
app.js - збирає всі функції в одну

Для того, щоб вигрузити галп та використовувати для іншого проекта треба зкопіювати: папки (gulp,src), файли(gulpfile, package.json)
далі по команді 'npm i' встаноляться всі плагіни
